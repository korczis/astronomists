## KEY PRINCIPLES

1. **Not repetition. Not decoration. Mythology.**
   - Every visual element carries narrative weight
   - Motifs combine to create complex meanings
   - Each touchpoint invites deeper understanding

2. **Minimalist but dense.**
   - 40%+ negative space (void as design element)
   - 3-color primary palette (maximum constraint)
   - Topology as visual backbone (mathematical, not organic)

3. **Systemic not arbitrary.**
   - Design system specifications (colors, typography, components)
   - Audit framework to validate compliance
   - CI/CD integration to enforce standards

4. **Grounded in place + reaching beyond.**
   - Czech cultural identity (Kepler, Cubism, Einstein)
   - Global scope (reaching toward 2100)
   - Local schools (starting with children)
