Astronautisté is not just a visual brand. It's a **mythology system**—a coherent narrative framework that explains *why* we look up, *how* we reach together, and *what* endures when everything else is noise.

This directory contains the complete architecture:

- **Framework**: The philosophical and narrative foundation
- **Design System**: Colors, typography, components, layout rules
- **Motif Library**: 10 archetypal visual concepts with expression variants
- **Export Guide**: Integration pathway via `/inject` skill

## CORE CONCEPT

### The Evolution: From Repetition to Mythology

**Phase 1: Repetitive Loop (Problem)**
- Every touchpoint looks the same: rockets, telescopes, slogans
- Visual fatigue sets in immediately
- No narrative depth
- No system

**Phase 2: Motif-Based (Current)**
- 10 distinct archetypal concepts, each with mythological weight
- Each appears in intentional sequence (not random)
- Paired with 3 meta-narratives (WITNESS, CONTINUITY, SIGNAL)
- Creates coherent narrative progression

**Phase 3: Minimalist Systemic (Target)**
- From decorative imagery to structural topology
- Colors: disciplined 3-color palette
- Typography: powerful, precise, hierarchical
- Layouts: 40%+ negative space (void as design element)
- Format: High-resolution 16:9 (professional standard)

## THE 3 META-NARRATIVES

### 1. WITNESS: The Observer's Journey
*"The moment you look, you belong."*

- Combines: Observer's Paradox + Dangerous Influencer
- Story: Child looks up → recognizes persistence → transforms from consumer to witness → develops epistemic humility
- Tone: Awe without sentimentality. Quiet power.
- Visual: Single point of light → horizon curve → standing figure reaching → integrated ownership

### 2. CONTINUITY: The Generational Relay
*"Every generation stands taller."*

- Combines: Evolutionary Arc + Motherhood Paradox + Czech Identity + Generation 2100
- Story: Past foundation → Present responsibility → Future achievement → Each generation reaches higher
- Tone: Hope grounded in physics. Ambition grounded in inheritance.
- Visual: Timeline showing ascending capability, silhouettes growing taller, bridges connecting generations

### 3. SIGNAL: Truth at Scale
*"Signal endures. Noise fades."*

- Combines: Observer's Paradox + Dangerous Influencer + Czech Identity in Orbit
- Story: Noise (algorithm chaos) → Signal (undeniable cosmos) → Integration (learning to listen) → Action (building systems that amplify signal)
- Tone: Intellectual clarity. Epistemic confidence. Anti-hype rigor.
- Visual: Left chaos (fractured lines, busy patterns) → Right clarity (single stars, geometric order)

## DESIGN SYSTEM AT A GLANCE

### Colors (3-Color Primary Palette)

| Name | Hex | Usage |
|------|-----|-------|
| **Void Black** | `#0a0a0a` | Primary background, cosmic depth |
| **Presence White** | `#f8f8f8` | Typography, primary content |
| **Signal Red** | `#e63946` | Emphasis, importance, action |

### Secondary/Functional Colors

| Name | Hex | Usage |
|------|-----|-------|
| **Prague Gold** | `#d4a574` | Warmth, heritage connection |
| **Space Blue** | `#1d3557` | Depth, credibility, secondary |
| **Growth Green** | `#06a77d` | Positive, progress, health |

### Typography

- **Sans-serif** (Inter, system-ui): Headlines, body, UI
- **Serif** (Georgia): Philosophical callouts, quotations
- **Monospace** (JetBrains Mono): Code, metrics, data

### Layout Rules

- ✓ No gradients | ✓ Minimum 40% negative space
- ✓ Sharp edges (0px border-radius on primary) | ✓ Maximum 1200px width
- ✓ Topology as backbone (orbits, networks, paths) | ✓ Geometric precision

## 5-LEVEL DESIGN HIERARCHY

### Level 1: Cosmic (Strategic)
**Scale**: Billboards, stages, installations | **Audience**: Global movement | **Focus**: Brand identity

### Level 2: Institutional (Organizational)
**Scale**: Educational posters, partnership collateral | **Audience**: Schools, research institutions | **Focus**: Access, inclusion

### Level 3: Product (Operational)
**Scale**: Web UI, mobile app, dashboards | **Audience**: End users | **Focus**: Data visualization metaphors

### Level 4: Community (Engagement)
**Scale**: Social media, downloadable assets | **Audience**: Users, educators, parents | **Focus**: Accessibility, shareability

### Level 5: Craft (Artistic)
**Scale**: Art books, limited-edition prints, merchandise | **Audience**: Collectors, cultural institutions | **Focus**: Heritage, artisanal quality

## IMPLEMENTATION TIMELINE

### Phase 1: Foundation (Weeks 1-2)
- Review framework with stakeholders
- Approve color palette + typography
- Prioritize motif production order
- Create design brief for illustrator

### Phase 2: Illustration Production (Weeks 3-6)
- Produce 20 SVG illustrations (Motifs 1-5, 4 expressions each)
- Quality assurance (accessibility, scalability, brand consistency)
- Export variations (SVG + PNG @ 1x/2x)

### Phase 3: Library Completion (Weeks 7-10)
- Produce remaining 20-40 illustrations (Motifs 6-10)
- Create animated sequences (3-4 key motifs)
- Generate topology/data visualization templates

### Phase 4: Integration (Weeks 11-12)
- Create HTML/CSS templates for all 5 design levels
- Generate Figma library + components
- Set up Storybook component stories
- Deploy `/inject` skill integration

### Phase 5: Testing & Audit (Week 13)
- Test across 5 use cases
- Run audit on sample projects
- Gather team feedback
- Launch documentation
