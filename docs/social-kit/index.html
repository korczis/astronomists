Náhled pro konkrétní stránku

Kterákoli stránka nebo sekce může mít vlastní náhled. Stačí do front matteru (.md) doplnit:

[extra]
og_image = "/og/social/generated-20260616/astronautiste-mission-control-classroom-wide.jpg"
og_image_alt = "Astronautisté — každé dítě na orbitu."

Bez tohoto override platí default výše. Šablona automaticky nastaví správný og:image:type podle přípony.

# přepíše data/og-previews.toml (zdroj této stránky)
python3 scripts/build-og-manifest.py
