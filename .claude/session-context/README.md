# Session Context

Append-only log of work sessions on this repo. One file per session, named
`YYYY-MM-DD-<slug>.md`. Synthesized by the `session-context-synthesizer` agent.

## Convention

- **Filename:** `YYYY-MM-DD-short-kebab-slug.md`
- **Content:** co se dělalo, proč, klíčová rozhodnutí, co zbývá. CZ rationale, EN technical.
- **Don't delete** old entries — append new ones.

See `.claude/protocols/CONTEXT-MANAGEMENT-PROTOCOL.md` for the full protocol.
