## VI. MESSAGING AUDIT CHECKLIST

### Pre-Launch Review (Every Post/Email/Asset)

- [ ] Does this claim pass NCLB (No Claim Left Behind) audit?
- [ ] Is the tone consistent with Tone of Voice guide (5 pillars)?
- [ ] Does this message reach the intended persona?
- [ ] Is the CTA clear + low-friction?
- [ ] Did we avoid forbidden claims?
- [ ] Is there evidence to back up any quantitative claim?
- [ ] Does the copy reflect the 10 design motifs (implicitly)?
- [ ] Did we avoid hype + buzzwords?
- [ ] Is the message persona-appropriate (formal for CISO, entrepreneurial for Founder)?
- [ ] Did we test this against competing messages?

---

### Monthly Governance Review

- [ ] Audit all live messaging (website, email, social)
- [ ] Collect engagement metrics (CTR, open rates, engagement)
- [ ] Identify underperforming messages + variants
- [ ] Test 3 new taglines or headlines
- [ ] Update case studies if new customer wins
- [ ] Check for claim decay (verify all evidence still current)
- [ ] Analyze competitor messaging (are we differentiating?)
- [ ] Gather customer feedback (resonance with real users)
