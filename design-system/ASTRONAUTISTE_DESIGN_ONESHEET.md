## TYPE SCALE (USE THESE SIZES)

```
TITAN       3.5rem  Caloto 700   | Hero headlines, max impact
HERO        2.8rem  Caloto 700   | Section titles, CTAs
DISPLAY     2.0rem  Caloto 600   | Subsections
TITLE       1.5rem  Inter 500    | Card titles
BODY        1.0rem  Inter 400    | Paragraph text, UI
SMALL       0.875rem Inter 500   | Captions, metadata
TINY        0.75rem Inter 400    | Timestamps, badges
MONO        0.875rem JetBrains Mono | Code, data output
```

**Key rules**:
- Caloto for headlines (geometric precision + warmth)
- Inter for body (legible, friendly, scales perfectly)
- JetBrains Mono for code/data (intelligence output)

## DO'S & DON'TS AT A GLANCE

```
вң… DO                              вқҢ DON'T

вң“ Use Pale Blue + Navy duet       вң— Mix Pale Blue + Gold in body copy
вң“ Nest icons in hexagons          вң— Use naked icons in dense layouts
вң“ Rule-of-thirds for heroes       вң— Stretch content to full width
вң“ 8px whitespace rhythm           вң— Use <8px spacing (looks cramped)
вң“ Max 3 color accents per page    вң— Use all 10 colors in one mockup
вң“ Hexagon grid on landing pages   вң— Hexagon grid on mobile dashboards
вң“ Blue + Navy for trust-building  вң— Red + Gold for primary CTAs
вң“ 16px minimum icon size          вң— Icons smaller than 16px (blurry)
вң“ Semantic colors for status      вң— Use gold for errors or warnings
вң“ 2px outline on focus states     вң— Browser default blue focus ring
```

## ACCESSIBILITY CHECKLIST

```
вҳҗ Color contrast ratios
  в”ңв”Җ Body text: 4.5:1 minimum (AAA)
  в”ңв”Җ Large text: 3:1 minimum (AA)
  в””в”Җ Test: WebAIM Contrast Checker

вҳҗ Focus states
  в”ңв”Җ All interactive elements: visible outline
  в”ңв”Җ Style: 2px solid Pale Blue, 2px offset
  в””в”Җ Never remove focus ring without replacement

вҳҗ Typography
  в”ңв”Җ Minimum 16px on mobile
  в”ңв”Җ Line-height: 1.5 for body text
  в””в”Җ Readable on 200% zoom

вҳҗ Icons
  в”ңв”Җ All images: descriptive alt text
  в”ңв”Җ Icons as links: aria-label
  в””в”Җ Status icons: color + pattern (not color alone)

вҳҗ Motion
  в”ңв”Җ Respect prefers-reduced-motion
  в”ңв”Җ Remove animations if set
  в””в”Җ Test: DevTools > Rendering > Emulate CSS media

вҳҗ Keyboard navigation
  в”ңв”Җ Tab order: logical (topвҶ’down, leftвҶ’right)
  в”ңв”Җ Visible skip links
  в””в”Җ No keyboard traps (arrow in modal, exit with Esc)
```
